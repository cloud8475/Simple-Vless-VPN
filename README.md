# Simple VPN

Набор bash-скриптов для быстрой установки и управления `VLESS + Reality` на `Xray` с раздачей подписки через `nginx`.

В проекте сейчас два основных файла:

- `setup.sh` — установка и первичная настройка сервера
- `manage.sh` — интерактивное управление сервером и пользователями

## Что делает `setup.sh`

Скрипт:

- устанавливает зависимости
- ставит `xray-core`
- генерирует `UUID`, ключи `Reality` и `shortId`
- создаёт конфиг `Xray`
- поднимает `nginx` для выдачи подписки
- создаёт файл с данными клиента

Во время запуска скрипт спрашивает локацию сервера:

- `Location icon`
- `Location name`

Из них собирается отображаемое имя сервера, например:

- `🇩🇪 Германия`
- `🇺🇸 USA`
- `🇯🇵 Tokyo`

Значения `de` и `Германия` показаны только как пример. Можно ввести любую иконку, emoji, код страны или произвольное название.

## Установка

Запускать на Ubuntu `20.04`, `22.04` или `24.04` от `root`:

```bash
sudo bash setup.sh
```

Во время установки скрипт задаст вопросы по локации. Если просто нажать `Enter`, будут использованы значения по умолчанию.

После завершения скрипт покажет:

- `Subscription URL`
- `VLESS Link`
- путь к файлу с данными клиента

## Что создаёт установщик

Основные файлы и пути:

- `/usr/local/etc/xray/config.json`
- `/usr/local/etc/xray/client-credentials.txt`
- `/var/www/sub/sub.txt`
- `/var/log/xray/access.log`
- `/var/log/xray/error.log`

## Управление сервером

Для управления используй:

```bash
sudo bash manage.sh
```

В меню доступны:

- запуск, остановка и перезапуск `Xray` и `nginx`
- просмотр статуса
- просмотр и выгрузка логов
- показ ссылки и QR-кода
- список пользователей
- добавление и удаление пользователей
- резервная копия конфига
- обновление `xray-core`
- просмотр трафика в реальном времени

## Пользователи

При добавлении нового пользователя через `manage.sh` скрипт:

- добавляет клиента в `config.json`
- перезапускает `xray`
- генерирует новую `VLESS`-ссылку

Название локации для новых ссылок берётся из `client-credentials.txt`, поэтому оно будет совпадать с тем, что было указано при установке.

## Важно

- Скрипты рассчитаны на запуск с `root`-правами.
- Для подписки используется `HTTP` на порту `8080`, если не задан другой `SUB_PORT`.
- Порт `VLESS` по умолчанию `443`.
- Название профиля подписки сейчас фиксировано как `Simple VPN`.

(проект создан для того чтобы просто настроить сервер для использования под VPN, сильно гибких настроек нету)
